# Mappings for determining governments for HoI4 countries.
#	They are evaluated in order until one applies, then that ideology and sub-ideology are applied to the country.

# Each mapping has the following elements:
#	vic_government (optional) - the government in Vic2. If the country has a different government the rule is skipped.
#	vic_tag (optional) - the Vic2 tag. If the country has a different tag the rule is skipped.
#	ruling_party (optional) - the ideology of the Vic2 party currently ruling the country.
#		If the country is ruled by a party with a different ideology the rule is skipped.
#	hoi_gov (required) - the ideology that will be ruling the HoI4 country.
#	hoi_leader (required) - the sub-ideology that the ruler of the HoI4 will hold.

# The file is organized into a few sections:
#	Main forms - the ideologies that will apply if all ideologies will be converted
#	CoE Main forms - the ideologies that will apply if all ideologies will be converted, with extras for Concert of Europe
#	Neutrality Forms - the neutral-form ideologies that will apply when ideologies have been dropped
#	CoE Neutrality Forms - the neutral-form ideologies that will apply when ideologies have been dropped, with extras for Concert of Europe
#	Limited Forms - when only default ideologies are converted, these rules will replace the converter's new ideologies
#	CoE Limited Forms - when only default ideologies are converted, these rules will replace the converter's new ideologies, with extras for Concert of Europe
